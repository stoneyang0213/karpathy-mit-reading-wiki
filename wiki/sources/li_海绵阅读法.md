## 元数据

- **Confidence: VERIFIED**(2026-04-16 stoneyang raw 采样:目录 + 前言 + 第四章核心方法 + 第五章知识体系 + 第六章致用/变现)

## 核心发现(validation 后)

### 1. "海绵"不是营销包装,是方法论立场
- 原名"吸血鬼读书法训练营",出版改名"海绵阅读法"
- 核心理念一致:"最大程度吸收书的精华,最大程度从阅读中受益"
- 不是说速度,是说**吸收率** — 阅读成果不是"读了多少本",是"受益程度有多深"

### 2. 七大能力框架(全书骨架)
| 章 | 能力 | 对应 wiki 概念 |
|---|---|---|
| 1 | 高效做读书笔记 | 三层笔记(碎片化 → 全局概览 → 结构内化) |
| 2 | 独立思考 | 想法的产生 → 判断 → 整理 |
| 3 | 掌握阅读速度 | 5 个"速度杀手" + "阅读是不是越快越好" |
| 4 | 阅读不同书籍 | 实用类 / 哲学社科 / 小说 / 历史(含"混读"洞见) |
| 5 | 建立知识体系 | 明确引用 Adler 主题阅读 |
| 6 | 让读书有用 | 五大原则 + 读书变现金字塔(4 层) |
| 7 | 长期持续稳定阅读 | 习惯养成四大定律 |

### 3. 六阶段阅读成长路径(自传式)
散漫读书 → 大量乱读 → 强制输出 → 建立知识体系 → 公众号写作变现 → 形成系统方法论

关键转折点:
- 第三阶段"强制输出"规则:"每看完一本书必须写一篇读书文章,如果没写就不允许看下一本"
- 核心追问:"读这本书,和没读这本书,对我来说有什么区别?"
- 第四阶段:从 Adler《如何阅读一本书》学到**主题阅读**,从"追求阅读量"转向"追求知识体系完整"

### 4. 明确的 Adler 师承关系
原文:"莫提默·J.艾德勒和查尔斯·范多伦的《如何阅读一本书》,我在这本书里学到了'主题阅读'这种方法"
- 她是 Adler 方法论在**中文大众实践端**的传承者
- 与王荣生(学术端)形成互补:王荣生面向语文教师,李小墨面向普通读者

### 5. 读书变现金字塔(4 层)
1. 写手级变现
2. 自媒体级变现
3. 知识付费产品经理级变现
4. 平台级变现

### 6. "混读"产生意外洞见
《别想那只大象》+《桃花庵歌》的例子 — 跨书跨领域阅读产生非线性洞见。
与宋家博"[[concepts/野蛮关联_候选|野蛮关联]]"是**同一现象的不同命名**。

## Debate 定位

- **Debate 02(获得 vs 成为)**:**获得为框架,成为是副产品**。主基调"让读书有用" + 变现金字塔,但也说"我的阅读史就是我的思想成长史和能力发展史",金钱是"意料之外的收获"。不是纯功利派。
- **Debate 03(快 vs 慢)**:**中间派**。专章讨论"阅读是不是越快越好",说"阅读速度和阅读质量同样重要"。
