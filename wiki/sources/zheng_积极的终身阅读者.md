# 郑钢《积极的终身阅读者》

## 元数据
- 作者:郑钢(同《怎样让学生爱上阅读》)
- **Confidence: LOW**

## 推测核心(v0,必须验证)

1. **核心主题**:如何从"被动读者"变成"终身积极阅读者"
   - 强调持续性 / 习惯 / 内在动力
   - 与 Adler 的"分析阅读技法"不同,更关注**态度和习惯**

2. **可能的方法论**
   - 阅读习惯的建立(时间 / 空间 / 仪式)
   - 克服阅读障碍(注意力 / 拖延 / 焦虑)
   - 阅读社群的作用
   - 终身学习者的自我认同

3. **定位**
   - 面向成年读者 / 教师 / 家长
   - 与《怎样让学生爱上阅读》形成"师生版"(这本对大人,那本对学生)

## 贡献给的框架
- [[11_终身阅读习惯]] — 候选,强调持续性而非单次阅读技法
