# 争议 02:阅读是为了"获得"还是"成为"?

> **Confidence: HIGH**(经典哲学分歧,跨中西方都有明确立场代表)

## 争议核心

**你为什么读书?**

### 实用主义派:阅读是为了"获得"
- 目标:获取知识 / 方法 / 信息 / 可以变现的能力
- 成功标准:读完这本书我能做什么新的事情?能赚更多钱 / 更好沟通 / 升职?
- 核心隐喻:**海绵 / 工具 / 弹药库**
- 方法:效率优先,笔记重点,输出驱动,不浪费时间

### 修养主义派:阅读是为了"成为"
- 目标:成为一个更深 / 更敏感 / 更自由的人
- 成功标准:读完这本书我变成了什么样的人?
- 核心隐喻:**避难所 / 觉醒 / 斧头 / 避风港**
- 方法:慢读,重读,留白,允许被干扰,允许不懂

### 实用主义派(以"获得"为主框架)
- [[sources/li_海绵阅读法]] — **VERIFIED(2026-04-16)**:主基调"让读书有用" + 读书变现金字塔(4层),但不纯功利 — "我的阅读史就是我的思想成长史",金钱是"意料之外的收获"。准确定位:**获得为框架,成为是副产品**
- [[sources/song_AI时代阅读力]] — **MEDIUM-VERIFIED**:温和同盟,"AI 可以成为深度阅读的高效工具",均衡快慢
- Adler 的前 3 层(检视 / 分析)— 实用倾向

### 修养主义派
- [[sources/mao_阅读是一座随身携带的避难所]] — "避难所"隐喻,毛姆的典型态度
- [[sources/hesse_阅读是安静的自我觉醒]] — "觉醒"隐喻,反功利主义
- [[sources/kafka_砍向我们内心冰封大海的斧头]] — "斧头"隐喻,最激进
- Adler 的第 4 层(主题阅读)— 在修养派一侧,追求"成为一个主题的思考者"

## 第三条路(Adler 的隐含立场)

Adler 其实是**两条路的双修派**:
- 前 3 层服务于"获得"(方法 / 技能 / 知识)
- 第 4 层主题阅读服务于"成为"(形成自己独立的论述)
- 他不反对实用,但最终指向修养
