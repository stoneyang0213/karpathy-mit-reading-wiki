# 03 文体决定读法

> **Confidence: HIGH**(Adler 的经典二分法,后续书都会讨论)

## 框架核心

不同文体需要不同的阅读策略,因为它们对心智的诉求不同。这是 14 本书语料库里**共识度最高**的框架之一。

### Adler 的原始二分(1940)

| 维度 | 论说性作品 | 想象文学 |
|---|---|---|
| 目标 | 传达知识 | 阐述经验 |
| 读者得到 | 经验中曾有/未有的知识 | 借阅读才能拥有的经验 |
| 方法 | [[02_分析阅读三层]] | 追求"深沉经验",去除阻碍 |
| 找什么 | 共识 / 主旨 / 论述 | **不要**找共识 / 主旨 / 论述 |

### 中国语文教育的细化(推测,王荣生一派)

中文语文教学论倾向于**更细分的文体分类**:
- 论说性 → 议论文 / 说明文
- 想象文学 → 小说 / 散文 / 诗歌 / 戏剧

每种文体的阅读教学内容都不同。这是对 Adler 二分法的细化而非反驳。

### Foster 的反叛(2003)

Foster 的《如何阅读一本文学书》**直接挑战 Adler 在想象文学上的立场**:
- Adler 说:想象文学不要找主旨 / 论述
- Foster 说:文学恰恰要训练"文学教授式"的符号识别 / 模式捕捉 / 互文理解

这是 14 本书里**最尖锐的争议**。见 [[debates/01_文学是否需要结构化分析]]

## 各书贡献

### Adler《如何阅读一本书》
原始二分法的提出者。立场:严格区分,各用各法。

### Foster《如何阅读一本文学书》
挑战 Adler 在文学端的立场,主张文学也需要结构化训练。

### 王荣生《阅读教学教什么》(待验证)
推测:支持"文体决定教学内容",但细分颗粒度比 Adler 更细(中文语文教学的传统)。

### 邵毅平《如何阅读文学经典》(VERIFIED 2026-04-17)

**不是讨论"古典文学 vs 现代文学"的文体区分,而是给出了一套"**中国古典文本的六种文脉操作**"**:

| 操作 | 示例 | 对应国际传统 |
|---|---|---|
| **空间解读** | 把《江南》放进"秦汉大一统四方意识" | Foster 互文 |
| **同题文脉** | 《闲情赋》→ 司汤达 → 钱钟书 → 林奕含 | Adler 主题阅读 |
| **历时演变** | 中国文学风景变迁 | Foster 模式识别 |
| **跨文体同题** | 跨越文体追踪同一主题(第五讲) | 邵毅平独创 |
| **知人论世** | 《项脊轩志》必须结合归有光生平 | 中国古典注疏传统 |
| **文本细读** | 用日本史料解读(第七讲) | Foster close reading |

**关键洞察**:邵毅平不是"文体决定读法"的新版本,而是"**同一文本可以被六种文脉操作激活**"的演示。这比 Adler 二分法维度更多,但保留了 Adler 的核心要求("文本置于文脉中去看"就是 Adler 第四层主题阅读的中文版)。

**对 stoneyang claude.md 的启发**:ingest 时不一定按文体分 prompt,可以按**文脉操作**分 prompt —— "让 AI 做空间文脉 / 历时文脉 / 同题文脉..."。

## stoneyang 的定位

你的 AI 流水线应该**支持所有文体,但对不同文体用不同的 ingest prompt**:

- 论说性书:让 AI 提取 Adler 三层(结构 / 主旨 / 论证)
- 小说:让 AI 提取 Foster 类目(符号 / 原型 / 互文)
- 诗歌 / 散文:让 AI 做意象 / 节奏 / 情感谱系
- 古典文学(邵毅平路线):让 AI 提取典故 / 意境 / 互文

**这就是"文体决定读法"在 claude.md ingest 规则里的具体体现** — 你的 claude.md 可以按文体写不同的 ingest 协议。
